# СтройГрад — Сайт строительной компании

Готовый к продакшену многостраничный сайт строительной компании на чистом HTML, CSS и JavaScript. Никаких фреймворков, никаких зависимостей — просто залейте файлы на хостинг и готово.

---

## Структура файлов

```
stroygrad/
├── index.html      # Главная страница
├── services.html   # Услуги
├── doctors.html    # Команда
├── prices.html     # Цены
├── contact.html    # Контакты
├── style.css       # Все стили (один файл для всех страниц)
└── script.js       # Весь JavaScript (один файл для всех страниц)
```

## Страницы

### `index.html` — Главная
- Hero-блок с анимированными плавающими карточками
- Счётчики (340+ объектов, 16 лет, 98% клиентов) с анимацией при скролле
- Блок преимуществ (4 карточки с иконками)
- Превью услуг (3 карточки-ссылки)
- Превью команды (4 специалиста)
- Отзывы клиентов (3 карточки)
- CTA-баннер с кнопкой и номером телефона

### `services.html` — Услуги
- 6 карточек услуг: строительство домов, ремонт, проектирование, фундамент, отделка, кровля
- Каждая карточка открывает модальное окно с подробным описанием и ценами
- Закрытие модалки по клику на оверлей или клавише `Escape`

### `doctors.html` — Команда
- 6 специалистов: генеральный директор, главный инженер, дизайнер, начальник производства, менеджер, сметчик
- Для каждого: инициалы-аватар, роль, опыт, образование, достижения, сертификаты

### `prices.html` — Цены
- 5 переключаемых вкладок: Строительство / Ремонт / Отделка / Фундамент / Кровля
- Таблицы с ценами, единицами измерения и сносками
- CTA с призывом заказать смету

### `contact.html` — Контакты
- Адреса офиса и производственной базы
- Телефоны, email
- Часы работы (будни, суббота, воскресенье, горячая линия 24/7)
- Форма заявки с валидацией и маской телефона
- Встроенная карта (OpenStreetMap, без API-ключа)

## Функциональность (JavaScript)

| Функция | Описание |
|---|---|
| Scroll-тень navbar | Появляется при прокрутке страницы вниз |
| Бургер-меню | Открытие/закрытие, закрытие по клику вне меню |
| Scroll Reveal | `IntersectionObserver` — элементы плавно появляются при прокрутке |
| Счётчики | Анимированный счёт чисел при первом появлении блока в viewport |
| Модальные окна | Открытие, закрытие по оверлею и Escape |
| Табы цен | Переключение разделов прайс-листа |
| Валидация формы | Проверка имени, телефона, email, выбора услуги; подсветка ошибок |
| Маска телефона | Автоформат `+7 (XXX) XXX-XX-XX` при вводе |
| Состояние успеха | После отправки формы показывает экран «Заявка отправлена» |
| Fade-in страниц | CSS-анимация при каждом открытии страницы |

## Деплой

### Любой статический хостинг (Beget, Timeweb, REG.RU и др.)

1. Распакуйте `stroygrad.zip`
2. Загрузите все файлы из папки `stroygrad/` в корень `public_html` (или нужную папку) через FTP или файловый менеджер хостинга
3. Откройте сайт в браузере

### GitHub Pages

```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/stroygrad.git
git push -u origin main
```
Затем в настройках репозитория: **Settings → Pages → Branch: main → Save**.

### Netlify / Vercel

Просто перетащите папку `stroygrad/` на [netlify.com/drop](https://app.netlify.com/drop) — сайт появится онлайн за 10 секунд.

## Кастомизация

### Изменить контактные данные
Найдите по всем файлам и замените:
- `+7 (495) 555-01-23` — ваш номер телефона
- `info@stroygrad.ru` — ваш email
- `г. Москва, ул. Строителей, д. 42` — ваш адрес

### Изменить цветовую схему
Все цвета в одном месте — начало файла `style.css`, блок `:root { ... }`. Измените значения переменных, и цвета обновятся по всему сайту.

### Заменить карту
В `contact.html` найдите тег `<iframe>` и замените `src` на нужный embed-URL:
- **Яндекс.Карты:** конструктор на [yandex.ru/map-constructor](https://yandex.ru/map-constructor/)
- **Google Maps:** кнопка «Поделиться» → «Встроить карту»
- **OpenStreetMap:** [openstreetmap.org](https://www.openstreetmap.org) → «Поделиться» → «HTML»

### Подключить реальную отправку формы
В `script.js` функция `submitForm()` сейчас показывает экран успеха без реальной отправки. Чтобы форма отправляла данные, замените её тело на `fetch`-запрос к вашему бэкенду или сервису (Formspree, EmailJS, GetForm и т.д.):

```js
async function submitForm() {
  // ... валидация ...
  if (n && p && eOk && s) {
    const res = await fetch('https://formspree.io/f/ВАШ_ID', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        name:    document.getElementById('f-name').value,
        phone:   document.getElementById('f-phone').value,
        email:   document.getElementById('f-email').value,
        service: document.getElementById('f-service').value,
        message: document.getElementById('f-msg').value,
      })
    });
    if (res.ok) {
      document.getElementById('formContent').style.display = 'none';
      document.getElementById('formSuccess').classList.add('show');
    }
  }
}
```

## SEO

Каждая страница содержит:
- уникальные `<title>` и `<meta name="description">`
- ключевые слова в `<meta name="keywords">`
- семантическую разметку: `<nav>`, `<section>`, `<footer>`, `<h1>`–`<h3>`
- атрибуты `alt` на SVG-иконках через `title` где применимо
- хлебные крошки на внутренних страницах

Для дополнительной оптимизации рекомендуется добавить `sitemap.xml` и `robots.txt`.
